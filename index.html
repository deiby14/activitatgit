Bienvenido a la pagina git log [opcions] --oneline: Mostra l'historial de commits en una sola línia per cada confirmació, incloent l'ID i el missatge curt. --graph: Representa visualment la història de branques i fusions. --all: Mostra els commits de totes les branques. --author=: Filtra els commits realitzats per un autor específic. --since= i --until=: Mostren els commits realitzats en un rang de dates específic. -n : Limita el nombre de commits mostrats (per exemple, -n 5 mostra els últims 5 commits). --stat: Mostra un resum de fitxers modificats i les línies afegides/eliminades. -p: Mostra els canvis detallats (diff) per a cada commit.
